# Import Excel Fleksibel Edura V4.6

Fitur ini dibuat untuk Excel sekolah/yayasan yang sudah lama dipakai dan biasanya punya banyak tab, judul tidak standar, dan nama sheet tidak sama dengan template Edura.

## Cara pakai

1. Buka **Edura Admin**.
2. Masuk ke menu **Import Excel**.
3. Upload file Excel lama.
4. Pilih mode **Fleksibel / Scan Semua Sheet**.
5. Klik **Baca & Analisis**.
6. Cek tabel **Analisis Semua Sheet**.
7. Centang sheet yang ingin diimport.
8. Ubah mode sheet jika tebakan Edura kurang tepat.
9. Cek preview sheet aktif.
10. Klik **Terapkan Import**.

## Yang bisa dibaca

- Sheet karyawan walaupun namanya `SDM`, `Guru`, `Data Pegawai`, atau nama lain.
- Sheet jabatan/fungsional.
- Sheet golongan/gapok.
- Sheet potongan.
- Sheet payroll/slip/rekap gaji.

## Catatan

Import tetap harus dicek lewat preview. Payroll adalah data sensitif, jadi Edura tidak langsung menimpa data tanpa konfirmasi admin.
